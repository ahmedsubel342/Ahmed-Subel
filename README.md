# Lakshmipur Sadar Portal

This is a simple community portal for **Lakshmipur Sadar Upazila**.  
It includes:
- 🚓 Police Helpline numbers  
- 🚑 Ambulance Services  
- 🏥 Hospitals & Clinics  
- 📢 Latest Updates  

### How to Run
1. Open `index.html` in any browser  
2. Or upload this repo to GitHub and enable **GitHub Pages**
